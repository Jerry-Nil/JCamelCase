# JCamelCase
> 多种命名法切换

## Usage

select some words and press keys

* macOS `command` + `shift` + `u`

* Windows/Linux `alt` + `shift` + `u`

`hello_world` -> `helloWorld`

`helloWorld` -> `HelloWorld`

`HelloWorld` -> `hello_world`
